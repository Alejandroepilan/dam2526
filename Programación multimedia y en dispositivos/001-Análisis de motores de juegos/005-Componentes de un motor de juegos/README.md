## 1. Introducción breve y contextualización

En esta actividad he trabajado con CSS3D, aplicando perspectiva, profundidad y transformaciones 3D a elementos HTML.

La idea principal era entender cómo una interfaz plana puede convertirse en una escena tridimensional modificando propiedades como ``perspective``, ``rotateX``, ``rotateY`` y ``translateZ``.

Esto me ha permitido ver cómo funciona una “cámara” simulada en CSS y cómo se organizan los elementos dentro de un espacio 3D.

---

## 2. Desarrollo detallado y preciso

A partir del archivo base, ajusté la perspectiva general de la escena y modifiqué los ángulos de rotación para experimentar con distintos puntos de vista.

Añadí un control deslizante para cambiar la perspectiva en tiempo real y un botón que reubica las tarjetas en coordenadas 3D aleatorias. Además, implementé una cámara controlada con teclado, permitiendo mover la vista con W, A, S y D. Todo esto se gestionó mediante JavaScript cambiando transformaciones CSS3D dinámicamente.

---

## 3. Aplicación práctica

Con los cambios realizados, la escena ahora permite:

- Ajustar la perspectiva y ver cómo se aleja o acerca la cámara.

- Distribuir los elementos por el espacio 3D de forma aleatoria, apreciando mejor la profundidad.

- Rotar la escena con el teclado para observarla desde distintos ángulos.

Estos ejemplos ayudan a visualizar de forma inmediata cómo se comportan los elementos cuando se modifican sus transformaciones en un entorno tridimensional.

---

## 4. Conclusión breve

Este ejercicio me ha ayudado a entender mejor cómo funcionan las transformaciones 3D, la cámara y la perspectiva en CSS, conceptos clave en la creación de interfaces tridimensionales.

También he visto cómo combinar CSS3D con JavaScript para lograr escenas interactivas y dinámicas, reforzando los contenidos de la unidad sobre representación 3D en la web.
