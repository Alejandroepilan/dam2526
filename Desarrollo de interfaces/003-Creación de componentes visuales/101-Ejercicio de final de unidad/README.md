## 1. Introducción breve y contextualización

En esta actividad he creado una mini librería de componentes UI básicos utilizando HTML, Tailwind CSS y un poco de JavaScript. El objetivo es practicar la creación de controles reutilizables para formularios, algo habitual en interfaces de gestión y aplicaciones empresariales.

---

## 2. Desarrollo detallado y preciso

La librería incluye cuatro componentes simples:

- Input con label

- Input con icono

- Select personalizado

- Tag input básico

Tailwind se usa para los estilos sin escribir CSS manual. El JavaScript controla la parte interactiva: abrir/cerrar el select, cambiar el valor seleccionado y crear etiquetas dinámicamente al pulsar Enter. Todos los componentes están diseñados para ser fáciles de copiar y usar en otros formularios.

---

## 3. Aplicación práctica

Estos componentes pueden emplearse en formularios de clientes, productos o tareas. 

El input con icono sirve para dar una mayor explicación sobre de que es el campo. 

El select permite gestionar estados como activo, pendiente o inactivo. 

El tag input facilita añadir categorías o etiquetas rápidas.
